## Recent Releases

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix"]}>

## New Features

- Added `<Tabs>` and `<CodeGroup>` components for multi-platform examples
- Introduced `<Columns>` layout for feature grids with responsive design
- Enhanced `<Steps>` with custom icons from Lucide library

## Improvements

- Optimized Mermaid diagram rendering for faster load times
- Improved accessibility with better alt text handling in `<Image>` components

## Bug Fixes

- Fixed parsing errors from unescaped `{` and `}` in prose text
- Resolved nested code block issues using 4-backtick fences

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "improvement"]}>

## New Features

- Integrated `<Update>` component for streamlined changelog entries
- Added `<ParamField>` and `<ResponseField>` for API documentation

## Improvements

- Brand color integration (`#3B82F6`) across all Callout and Card components
- Better heading hierarchy enforcement starting from H2 only

## Bug Fixes

- Corrected JSX attribute syntax to enforce double quotes everywhere

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["breaking", "feature"]}>

## New Features

- Launched core MDX components: `<Card>`, `<Callout>`, `<Steps>`
- Full support for Mermaid code blocks (no JSX component needed)

## Breaking Changes

- Removed H1 headings from body content; use frontmatter title instead

## Improvements

- Strict parsing rules prevent common errors like raw `{variable}` in text

</Update>
